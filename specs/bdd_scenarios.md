# CareerOS AI - BDD Scenarios

## Scenario 1: Career Roadmap Generation

Given a student provides career goals

When the Career Strategist Agent analyzes the profile

Then a personalized roadmap should be generated

And recommended skills should be provided

---

## Scenario 2: Research Paper to Project

Given a student uploads a research paper

When the Research-to-Project Agent processes the document

Then a project idea should be generated

And a technology stack should be suggested

And implementation milestones should be produced

---

## Scenario 3: Hackathon Guidance

Given a student requests hackathon assistance

When the Hackathon Agent analyzes the request

Then suitable project ideas should be generated

And an execution plan should be created

---

## Scenario 4: Scholarship Discovery

Given a student provides academic information

When the Scholarship Agent evaluates eligibility

Then matching scholarship opportunities should be returned

And an application checklist should be generated

---

## Scenario 5: Internship Assessment

Given a student uploads a resume

When the Internship Agent evaluates the profile

Then an internship readiness score should be produced

And skill gaps should be identified

And improvement recommendations should be generated

---

## Scenario 6: MCP Tool Usage

Given an agent requires external information

When an MCP tool is invoked

Then the tool call should be logged

And the result should be returned to the requesting agent

---

## Scenario 7: Security Validation

Given an agent attempts to access a tool

When the tool is not in the allowlist

Then access should be denied

And the event should be logged
