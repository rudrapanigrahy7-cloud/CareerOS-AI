# CareerOS AI - Skills Specification

## Overview

Skills provide reusable procedural knowledge that agents can invoke when required.

Each skill contains:
- Instructions
- Best practices
- Examples
- Workflows

Skills are triggered only when relevant to a user request.

---

# 1. Career Strategy Skill

## Purpose

Create structured career roadmaps.

## Used By

Career Strategist Agent

## Outputs

- Learning roadmap
- Skill recommendations
- Milestones
